# NAOCacsi

Algoritmo em Python 2.x que identifica as cores azul, amarelo, vermelho e verde utilizando a biblioteca de visão computacional OpenCv.
Finalidade: Resumo estendido, CACSI 2015 / Fins acadêmicos.
